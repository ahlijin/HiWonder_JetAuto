# 0.9.3

## 更新内容

### 硬件配置简化

- 删除了所有 `MACHINE_TYPE` / `LIDAR_TYPE` / `DEPTH_CAMERA_TYPE` 环境变量判断
- URDF 文件直接包含 JetAutoPro / A1 / AstraPro 对应文件，无分支逻辑
- `odom_publisher_node.py`、`joystick_control.py`、`controller_manager.py`、`teleop_key_control.py` 中的机型判断代码全部清理
- `pi3_env.sh`、`jetson_env.sh` 精简为只保留 `ROS_DOMAIN_ID=42`

### Jetson 侧新增控制能力

- **新增 Jetson 键盘控制节点**：`jetauto_peripherals/teleop_key_control.py`，发布 `/cmd_vel` 控制小车移动
- **新增 Jetson 舵机控制工具**：`jetauto_peripherals/servo_control.py`，命令行发布到 `/servo_controller`
- 从 Pi3 侧复制 `servo_controller_msgs/` 到 Jetson 侧，支持舵机控制编译

### Pi3 资源优化

- **移除 IMU 滤波 + EKF**：`imu_filter_madgwick` 和 `robot_localization` 不再在 Pi3 上运行，里程计直接从 `odom_publisher` 发布到 `/odom`
- **移除重复的 robot_description 启动**：Pi3 `odom_publisher.launch.py` 不再包含 `robot_description_launch`
- **激光滤波移至 Jetson**：`laser_filters` 从 Pi3 的 `lidar.launch.py` 移至 Jetson 的 `lidar_receiver`
- **启动等待缩短**：`startup_check.py` 从 `time.sleep(30)` 改为 `time.sleep(2.0)`
- **手柄速度减半**：`max_linear` 从 0.5 降至 0.25，`max_angular` 从 2.0 降至 1.0

### TF 架构优化

- 移除 Jetson 侧冗余的 `robot.launch.py` TF 发布层
- `robot_state_publisher` 和 `joint_state_publisher` 只在 Pi3 侧运行
- 机器人模型参数通过 DDS 从 Pi3 传递到 Jetson

### 代码结构整理

- **删除 Jetson 侧 `jetauto_description`**（节省 ~30MB STL 模型文件，Pi3 统一发布）
- **`driver/` 重命名为 `sdk/`**：目录名与包名对齐
- **Jetson `peripherals` 更名为 `jetauto_peripherals`**：避免与 Pi3 同名包冲突
- **合并 `servo_controller.py` → `controller_manager.py`**：去掉不必要的抽象层
- **删除死代码**：`sdk/button.py`、`sdk/led.py`、`sdk/fps.py`、`sdk/pid.py`（依赖 Jetson.GPIO，Pi3 上不可用）
- **删除机械臂/夹爪相关代码**：`joint_trajectory_action_controller.py` 及 `controller_manager.py` 中对应的初始化逻辑，`joints` 列表精简为仅 `joint1`
- **清理 git 垃圾**：删除 `__pycache__`、`Log/*.log.txt`，更新 `.gitignore`

### 文档完善

- README 更新目录结构、功能说明、启动方式、依赖清单
- 新增 `Changelog/0.9.3.md`

## 启动方式变更

### Pi3

```bash
cd ~/pi3_ws
./pi3_start.sh
```

### Jetson

```bash
cd ~/jetson_ws

# SLAM 建图（后台，适合 ssh）
./jetson_start.sh

# SLAM 建图（桌面，含 RViz）
./jetson_slam.sh

# 键盘控制小车移动
./jetson_teleop.sh

# 舵机控制
ros2 run jetauto_peripherals servo_control <servo_id> <position>

# 导航
./jetson_navigation.sh
```

### 编译说明

Pi3 注意 `driver/` 已重命名为 `sdk/`：

```bash
cd ~/pi3_ws
rm -rf build/driver install/driver
colcon build
```

Jetson 注意包名从 `peripherals` 变更为 `jetauto_peripherals`：

```bash
cd ~/jetson_ws
rm -rf build/peripherals install/peripherals
colcon build --packages-select servo_controller_msgs jetauto_peripherals slam lidar_receiver
```
