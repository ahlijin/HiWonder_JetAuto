# 0.9.5

## 更新内容

### 启动蜂鸣修复

- `startup_check.py` 不再使用固定 `time.sleep(2.0)`，改为等待 `ros_robot_controller/init_finish` 服务就绪后再蜂鸣（超时 10 秒，超时则跳过并打 warning）
- 解决了 Pi3 启动后无蜂鸣提示音的问题

### 键盘控制优化

- **修复 Jetson 侧 `teleop_key_control.py` 按键松开后小车不停止的问题**：按下按键移动，松开按键自动发布零速 `Twist` 停止小车，无需额外按键

### 数据链路精简

- **删除 `lidar_bridge`（Pi3）和 `lidar_receiver`（Jetson）**：不再需要桥接转发节点，Pi3 的 `/scan_raw` 通过 DDS 直接暴露给 Jetson
- **`pi3_bringup.launch.py`**：去掉 `lidar_bridge` 节点
- **`jetson_bringup.launch.py` 和 `jetson_slam.launch.py`**：去掉 `lidar_receiver` 节点，滤波配置移至 `slam/config/lidar_filters_config.yaml`
- **数据流简化**：`Pi3: /scan_raw → DDS → Jetson: scan_raw → laser_filter → scan`

### 清理作者信息

- 删除所有 Python 文件中的 `@Author: Aiden` 和 `@Date` 注释（共 6 处）
- `setup.py` 和 `package.xml` 中的 `maintainer_email` 替换为占位邮箱（共 27 个文件）

### 编译说明变更

Pi3：

```bash
cd ~/pi3_ws
rm -rf build/lidar_bridge install/lidar_bridge
colcon build
```

Jetson：

```bash
cd ~/jetson_ws
rm -rf build/lidar_receiver install/lidar_receiver
colcon build --packages-select ros_robot_controller_msgs jetauto_peripherals slam
```
