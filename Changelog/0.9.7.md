# 0.9.7

## 更新内容

### 舵机控制修复

- **`ros_robot_controller_node.py`**：启动时自动使能总线舵机扭矩（`bus_servo_enable_torque(1, 1)`），避免舵机通电后处于 unloaded 状态不响应位置指令
- **`controller_manager.py`**：`set_position()` 中 `ServoPosition.position` 赋值由 `int` 改为 `float`，匹配 ROS2 消息类型定义
- **`controller_manager.py`**：`publish_joint_states()` 中 `ServoState.position` 保持 `int`，匹配 ROS2 消息类型定义
- **`ros_robot_controller_node.py`**：`set_bus_servo_position()` 中 `i.position` 转为 `int` 再传给 SDK，避免 `struct.pack` 因 float 参数报错

### 启动蜂鸣器修复

- **`startup_check.py`**：publish 蜂鸣器消息后增加 `time.sleep(0.5)`，给 DDS 足够时间投递消息，确保蜂鸣器在启动时正常响起

### 编译说明变更

Pi3 无新增依赖，无需重新编译，仅替换运行文件即可：

```bash
# 同步到 Pi3
scp pi3/src/ros_robot_controller/ros_robot_controller/ros_robot_controller_node.py pi@192.168.3.64:~/pi3_ws/install/ros_robot_controller/lib/python3.10/site-packages/ros_robot_controller/
scp pi3/src/servo_controller/servo_controller/controller_manager.py pi@192.168.3.64:~/pi3_ws/install/servo_controller/lib/python3.10/site-packages/servo_controller/
scp pi3/src/bringup/bringup/startup_check.py pi@192.168.3.64:~/pi3_ws/bringup/lib/python3.10/site-packages/bringup/
```
