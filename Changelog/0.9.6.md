# 0.9.6

## 更新内容

### Pi3 依赖精简与部署优化

- **删除 Pi3 侧 `sdk/` 包**：`sdk/common.py` 依赖 `cv2`、`numpy`、`yaml`、`transforms3d`，Pi3 上全部用不到
- **`joystick_control.py`**：`val_map` 直接内联到文件中，不再依赖 `sdk` 包
- **`peripherals/package.xml`**：去掉 `<depend>sdk</depend>`

### udev 规则修复

- **`lidar.rules`**：雷达 USB 芯片实际为 CH340（`1a86:7523`），修正错误的 CP2102（`10c4:ea60`）配置
- **恢复 `sllidar_ros2/scripts/rplidar.rules` 为原始内容**：不动外部代码
- **`99-ttyACM0.rules`**：修复末尾 `"1"1"` 语法错误

### 文档新增

- **`pi3/doc/pi3部署.md`**：Pi3 侧部署指南，包含 ROS 源配置、依赖安装、udev 规则配置、启动测试步骤

### 编译说明变更

Pi3：

```bash
cd ~/pi3_ws
rm -rf build/sdk install/sdk build/peripherals install/peripherals
colcon build --packages-select peripherals
colcon build
```
